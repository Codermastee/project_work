What is GeneNFT?

GeneNFT is a blockchain-based framework that tokenizes Raw Genomic Data (RGD) as NFTs. Each RGD NFT acts as a parent asset for Sequenced Genomic Data (SGD) NFTs, ensuring complete traceability and data lineage.

By integrating Threshold Cryptography, IPFS storage, and Fully Homomorphic Encryption, GeneNFT ensures privacy, scalability, and decentralized control.

Core Innovations

  • Composable Genomic NFTs
  • Threshold Cryptographic Key Sharing
  • IPFS Off-Chain Storage
  • FHE-Based Secure Computation
  • Automated Monetization Smart Contracts
