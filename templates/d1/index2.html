About GeneNFT

GeneNFT is a decentralized genomic data management platform that allows individuals to securely own, control, and monetize their genomic information using Blockchain, NFTs, IPFS, and Privacy-Preserving Cryptography. The system ensures that sensitive DNA data is never exposed while enabling trusted biomedical research.

System Workflow

1. Genome Upload

User uploads genomic data which is encrypted before storage.

2. Secure Storage

Encrypted genome stored in IPFS with immutable hash reference.

3. NFT Ownership

RGD NFT minted representing true ownership of genomic asset.

4. Privacy Analysis

Researchers compute insights using encrypted data without access to raw DNA.

System Actors

Data Owner

Individuals who upload genome data, control access permissions, and earn compensation from its usage.

Researcher

Authorized users who request access to perform privacy-preserving genomic analysis.

Blockchain Network

Ensures transparent ownership tracking, smart contract execution, and immutable audit history.
